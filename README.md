# MYHazard

MYHazard is a location-based mobile application that allows users to report and view nearby road hazards such as floods, accidents, potholes, and roadblocks.

## Features
- User registration and login
- Automatic location detection
- Submit road hazard reports
- View nearby hazards on Google Maps
- Severity-based hazard markers

## Tech Stack
- Android (Java)
- Retrofit & OkHttp
- Google Maps SDK
- Node.js & Express
- MongoDB Atlas
